# Kodo ShelfBot

An autonomous warehouse robot combining:

- Differential Drive AMR
- 6DOF Manipulator
- ROS 2 Navigation
- Perception & Grasping
- Docking & Shelf Picking

---

## Architecture Overview

This project explores full-stack warehouse robotics:

- Navigation stack (Nav2)
- Arm motion planning (MoveIt 2)
- Simulation in Gazebo
- Integrated task planning
